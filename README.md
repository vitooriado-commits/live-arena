# LIVE ARENA™ Core 7.1 Game Engine

Primeira versão do motor de jogo interativo.

## Inclui

- Battle Engine
- Boss HP
- Boss phases
- Energy system
- Combo system
- AI Director
- Dynamic Events
- Controls for testing

## Como usar

1. Envia o conteúdo do ZIP para GitHub.
2. Abre `index.html`.
3. Testa os botões no rodapé.
